# PetShare 🐾

一个可爱的宠物照片分享社区网站，让宠物爱好者们可以分享和欣赏可爱的宠物照片。

## 功能特点 ✨

- 📸 **照片上传**: 支持上传宠物照片并添加描述
- 🏷️ **宠物信息**: 添加宠物名字和介绍
- ❤️ **点赞功能**: 给喜欢的宠物照片点赞
- 💬 **评论系统**: 与其他宠物爱好者互动
- 📱 **响应式设计**: 完美适配手机和电脑
- 🎨 **现代UI**: 渐变色彩和毛玻璃效果

## 技术栈 🛠️

- HTML5
- CSS3 (渐变、动画、响应式布局)
- JavaScript (原生JS)
- 无需后端依赖

## 使用方法 🚀

1. 下载项目文件
2. 直接在浏览器中打开 `pet-share.html`
3. 开始分享你的宠物照片！
